Todo / Wishlist Web App
📌 Project Description

This project is a simple Todo / Wishlist Web Application built using HTML, CSS, and JavaScript.
Users can add tasks or wishes, mark them as completed, and delete them.
The application stores the data using Browser Local Storage, which keeps the tasks saved even after refreshing or reopening the page.

🚀 Features

Add new todo / wishlist items
Mark tasks as completed
Delete tasks from the list
Unique ID generated for each task
Tasks stored using Local Storage (JSON format)
Data persists after page refresh

🛠 Technologies Used

HTML
CSS
JavaScript
Browser Local Storage
JSON (for data storage format)

⚙️ How It Works
User enters a task in the input field
Task is added to an array of objects
Each task gets a unique ID
The array is converted to JSON using JSON.stringify()
Data is stored in Local Storage
When the page reloads, data is retrieved using JSON.parse()

▶️ How to Run the Project

Clone or download the repository
Open the project folder
Open index.html in your browser
Start adding your tasks
